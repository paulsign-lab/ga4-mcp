# 📊 주간 채널별 리포트

매주 월요일 아침 보는 채널 성과 리포트입니다.  
아래 질문을 Claude에 그대로 붙여넣으세요.

---

## 기본 (30초)

```
지난 7일 채널별 세션 수와 전환 수를 표로 정리해줘
```

---

## 심화 — 전주 비교 포함 (60초)

```
지난 7일과 그 전 7일을 비교해서 채널별 성과 표를 만들어줘.

표 항목:
- 채널
- 이번 주 세션 수
- 지난 주 세션 수
- 세션 변화율 (▲▼ 표시)
- 이번 주 전환 수
- 이번 주 전환율

마지막에 핵심 인사이트 2줄 추가해줘.
```

---

## 월간 버전

```
이번 달 채널별 세션 수, 전환 수, 전환율을 세션 많은 순으로 표로 정리해줘
```

---

## 활용 팁

- "전환 수 대신 매출로 바꿔줘" 를 추가하면 매출 기준 리포트로 변경됩니다
- "채널 대신 소스/매체로 보여줘" 를 추가하면 `sessionSourceMedium` 차원으로 변경됩니다
- 날짜 범위를 직접 지정하려면: "2026-06-01부터 2026-06-30까지 채널별..."
