# 📄 랜딩 페이지 분석

어떤 페이지로 들어오는 사람이 많은지, 어디서 이탈하는지 파악합니다.

---

## 기본 — TOP 10 랜딩 페이지

```
지난 28일 랜딩 페이지 TOP 10을 세션 수, 이탈률, 평균 체류시간 포함해서 표로 보여줘
```

---

## 전환율 기준 분석

```
지난 28일 랜딩 페이지별 전환율을 분석해줘.

표 항목:
- 랜딩 페이지 URL
- 세션 수
- 전환 수
- 전환율

전환율 TOP 5와 하위 5개를 각각 정리하고, 차이가 나는 이유를 추론해줘.
```

---

## 이탈률 문제 페이지 찾기

```
지난 28일 기준 세션 수 100회 이상인 랜딩 페이지 중
이탈률이 가장 높은 TOP 5를 찾아줘.
각 페이지별 이탈률과 평균 체류시간도 포함해줘.
```

---

## 활용 팁

- 특정 URL만 보려면: "URL에 '/product' 가 포함된 랜딩 페이지만 분석해줘"
- 신규 vs 재방문 비교: "위 결과를 신규 사용자와 재방문자로 나눠서 비교해줘"
