# 📅 기간 비교 분석

이번 주 vs 지난 주, 이번 달 vs 지난 달 성과를 비교합니다.

---

## 주간 비교 — 전체 트래픽

```
이번 주(최근 7일)와 지난 주(그 전 7일)를 비교해서
전체 세션 수, 전환 수, 전환율 변화를 보여줘.
변화율은 ▲▼ 기호로 표시해줘.
```

---

## 채널별 주간 비교

```
이번 주와 지난 주를 채널별로 비교해줘.

표 항목:
- 채널
- 이번 주 세션
- 지난 주 세션
- 변화율 (▲▼)
- 이번 주 전환율
- 지난 주 전환율

가장 많이 오른 채널과 가장 많이 떨어진 채널에 대한 인사이트 추가해줘.
```

---

## 월간 비교

```
이번 달과 지난 달을 비교해서
채널별 세션 수와 전환 수 변화를 표로 보여줘.
전월 대비 전체 성과 요약 1줄 추가해줘.
```

---

## 캠페인 기간 직접 지정

```
2026-06-01부터 2026-06-15까지와
2026-05-01부터 2026-05-15까지를 비교해서
채널별 세션과 전환 수 변화를 분석해줘.
```
