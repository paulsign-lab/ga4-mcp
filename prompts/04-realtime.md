# ⚡ 실시간 모니터링

광고 집행 직후, 콘텐츠 발행 직후 실시간 반응을 확인합니다.

---

## 지금 활성 사용자

```
지금 내 사이트에 활성 사용자 몇 명이야? 어떤 페이지 보고 있어?
```

---

## 채널별 실시간 유입

```
지금 활성 사용자를 채널별로 분류해서 보여줘.
어떤 채널에서 가장 많이 들어오고 있어?
```

---

## 광고 집행 직후 확인

```
지금 활성 사용자 현황과 어떤 페이지에서 사람들이 머물고 있는지 보여줘.
광고 랜딩 페이지 URL도 함께 확인해줘.
```

---

## 활용 팁

- 실시간 데이터는 현재 기준 약 30분 이내 데이터를 보여줍니다
- SNS 게시물 발행 직후 반응 확인에 유용합니다
- 이메일 발송 후 유입 급증 여부 체크에 활용하세요
