# 📋 종합 마케팅 리포트

경영진·팀장에게 공유하는 월간 또는 주간 종합 리포트입니다.  
아래 질문 하나로 리포트 전체를 자동 생성합니다.

---

## 주간 종합 리포트

```
[역할] D2C 브랜드 그로스 마케터

[기간] 지난 7일 (이번 주) vs 그 전 7일 (지난 주)

[요청] 아래 3개 표를 순서대로 만들어줘.

표 1. 채널별 성과
- 채널 / 이번 주 세션 / 지난 주 세션 / 변화율(▲▼) / 이번 주 전환 수 / 전환율

표 2. 랜딩 페이지 TOP 5
- URL / 세션 수 / 이탈률 / 평균 체류시간

표 3. 디바이스별 분포
- 디바이스 / 세션 수 / 전환율

[인사이트] 표 3개 아래에 핵심 인사이트 3줄 추가
- 이번 주 가장 눈에 띄는 변화 1줄
- 개선이 필요한 포인트 1줄
- 다음 주 집중해야 할 액션 1줄

[포맷] 한국어 표 헤더 / 비율 소수점 둘째 자리 / 수치 천 단위 쉼표
```

---

## 월간 종합 리포트

```
[역할] 퍼포먼스 마케터

[기간] 이번 달 vs 지난 달

[요청] 월간 성과 리포트를 작성해줘.

1. 전체 요약 (세션 / 신규 사용자 / 전환 수 / 전환율 — 전월 비교)
2. 채널별 성과 표 (세션 / 전환 / 전환율 / 전월 대비 변화율)
3. TOP 5 랜딩 페이지 (세션 / 이탈률)
4. 디바이스별 세션 비중

[인사이트]
- 이달 가장 성과가 좋았던 채널과 이유 추론
- 다음 달 예산 배분 제안 1줄

[포맷] 한국어 헤더 / 비율 소수점 둘째 자리 / 수치 천 단위 쉼표
```

---

## 경영진 보고용 요약 (1페이지)

```
지난 28일 GA4 데이터를 바탕으로 경영진 보고용 한 페이지 요약을 만들어줘.

포함할 내용:
- 핵심 지표 4개 (세션 / 전환 / 전환율 / 전월 대비 변화율)
- 채널별 기여도 비중 (%)
- 가장 성과 좋은 채널 + 가장 성과 낮은 채널 각 1개
- 다음 달 개선 제안 2줄

간결하게 작성하고, 숫자는 반드시 포함해줘.
```

## 활용 팁

- 매주 월요일 아침 이 프롬프트를 실행하면 주간 리포트 자동 완성
- "결과를 한국어 이메일 형식으로 정리해줘" 추가 시 팀 공유용 메일 초안 생성
- "매출 데이터도 포함해줘" 추가 시 수익 지표까지 포함 (전환 이벤트에 매출 설정돼 있어야 함)
