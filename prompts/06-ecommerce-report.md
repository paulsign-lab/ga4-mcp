# 🛒 B2C 이커머스 분석

온라인 쇼핑몰·제품 판매 사이트 전용 분석 템플릿입니다.  
`purchase` 또는 `totalRevenue` 이벤트가 GA4에 설정돼 있어야 합니다.

> 💡 이 파일을 사용하기 전에 `00-first-steps.md` → Step 4에서 전환 이벤트를 확인하세요.

---

## 매출 현황

### 기본 매출 리포트

```
지난 28일 채널별 매출과 구매 전환율을 표로 정리해줘.

표 항목:
- 채널
- 세션 수
- 구매 수
- 매출 (₩ 단위)
- 구매 전환율
- 세션당 매출 (₩)

매출 기여가 가장 큰 채널 TOP 3와 인사이트 2줄 추가해줘.
```

### 이번 달 vs 지난 달 매출 비교

```
이번 달과 지난 달의 매출, 구매 수, 평균 주문 금액(AOV)을 비교해줘.
변화율은 ▲▼ 기호로 표시하고, 변화의 원인을 추론해줘.
```

## 제품 성과

### 제품별 매출 TOP 10

```
지난 28일 제품별 매출 TOP 10을 아래 항목으로 표로 보여줘:
- 제품명
- 조회 수 (view_item)
- 장바구니 추가 수
- 구매 수
- 매출 (₩)
- 조회→구매 전환율
```

### 장바구니 이탈 분석

```
지난 28일 기준 장바구니 이탈 분석을 해줘.

각 단계별 수치:
- 제품 조회 (view_item)
- 장바구니 추가 (add_to_cart)
- 결제 시작 (begin_checkout)
- 구매 완료 (purchase)

각 단계 이탈률(%)도 계산해줘.
가장 많이 이탈하는 단계와 개선 방안 제안 1줄 추가해줘.
```

## 고객 분석

### 신규 vs 재구매 고객

```
지난 28일 기준 신규 고객과 재방문 고객을 비교해줘.

표 항목:
- 구분 (신규/재방문)
- 세션 수
- 구매 수
- 전환율
- 평균 주문 금액 (₩)

재방문 고객의 전환율이 신규보다 높은지 낮은지, 이유를 추론해줘.
```

### 디바이스별 구매 성과

```
지난 28일 디바이스별(모바일/데스크탑/태블릿) 구매 성과를 비교해줘.

표 항목:
- 디바이스
- 세션 수
- 구매 전환율
- 평균 주문 금액 (₩)

모바일 전환율이 낮다면 개선 방안 제안해줘.
```

## 캠페인·광고 성과

### UTM 캠페인별 ROAS

```
지난 28일 UTM 캠페인별 성과를 표로 정리해줘.

표 항목:
- 캠페인명
- 세션 수
- 구매 수
- 매출 (₩)
- 구매 전환율

ROAS 계산을 위해 광고비 입력이 필요하면 알려줘.
매출 기준 상위 3개 캠페인 인사이트 추가해줘.
```

### 채널별 신규 고객 획득 현황

```
지난 28일 채널별로 신규 사용자 수와 첫 구매 전환율을 비교해줘.
고객 획득 효율이 가장 높은 채널이 어디인지 알려줘.
```

## 종합 이커머스 리포트

경영진·팀장에게 보고할 때 사용하세요.

```
[역할] 이커머스 그로스 마케터

[기간] 지난 28일 vs 그 전 28일

[요청] 이커머스 종합 리포트를 작성해줘:

1. 핵심 지표 요약 (전월 비교)
   - 총 매출 / 구매 수 / 평균 주문 금액 / 구매 전환율

2. 채널별 매출 기여 표
   - 채널 / 매출 / 구매 수 / 전환율 / 전월 대비 변화율(▲▼)

3. 제품 TOP 5
   - 제품명 / 판매 수 / 매출

4. 장바구니 이탈 퍼널
   - 조회→장바구니→결제→구매 단계별 이탈률

5. 핵심 인사이트 3줄
   - 가장 잘 된 것 / 문제 포인트 / 다음 달 액션

[포맷] 한국어 헤더 / ₩ 단위 / 비율 소수점 둘째 자리
```

## 활용 팁

- **GA4에 `totalRevenue`가 0으로 나오는 경우**: GA4 전자상거래 이벤트(`purchase` + `value` 파라미터) 설정을 확인하세요
- **제품 데이터가 안 나오는 경우**: GA4 향상된 전자상거래 또는 `items` 배열 설정이 필요합니다
- **ROAS 계산**: GA4는 광고비를 직접 집계하지 않아 채널별 매출만 제공합니다. 광고 플랫폼 비용과 수동으로 비교하거나 Google Ads MCP를 함께 사용하세요
