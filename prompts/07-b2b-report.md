# 🏢 B2B 기업 웹사이트 분석

제품 소개·카탈로그 다운로드·문의 폼이 있는 기업 웹사이트 전용 분석 템플릿입니다.

> 💡 이 파일을 사용하기 전에 `00-first-steps.md` → Step 4에서 어떤 전환 이벤트가  
> 설정돼 있는지 먼저 확인하세요. B2B 사이트는 전환 이름이 회사마다 다릅니다.

---

## B2B 사이트에서 '전환'이란

B2B 웹사이트에서는 매출 대신 **리드(잠재 고객 접촉)**가 핵심 지표입니다.

| 전환 유형 | GA4 이벤트 이름 (보통) |
|---|---|
| 문의 폼 제출 | `generate_lead` `form_submit` `contact_form` |
| 카탈로그·브로슈어 다운로드 | `file_download` `download_catalog` |
| 견적 요청 | `request_quote` `rfq_submit` |
| 제품 상담 예약 | `book_demo` `schedule_meeting` |
| 뉴스레터 구독 | `newsletter_signup` |

> 💡 내 사이트의 이벤트 이름이 다를 수 있습니다. `00-first-steps.md` → Step 4로 확인하세요.

## 리드 발생 현황

### 기본 리드 리포트

```
지난 28일 채널별 리드(전환) 발생 수와 리드 전환율을 표로 정리해줘.

표 항목:
- 채널
- 세션 수
- 리드 수 (전환 수)
- 리드 전환율
- 세션 대비 리드 기여율(%)

리드가 가장 많이 발생하는 채널 TOP 3와 인사이트 2줄 추가해줘.
```

### 리드 소스 심화 분석

```
지난 28일 UTM 소스·매체별 리드 발생 현황을 분석해줘.

표 항목:
- 소스/매체
- 세션 수
- 리드 수
- 리드 전환율

어떤 매체에서 유입된 사람들이 가장 높은 리드 전환율을 보이는지 알려줘.
```

## 카탈로그·자료 다운로드 분석

```
지난 28일 파일 다운로드 이벤트(file_download)가 발생한 페이지를 분석해줘.

표 항목:
- 다운로드 발생 페이지
- 다운로드 수
- 해당 페이지 세션 수
- 다운로드 전환율(%)

가장 많이 다운로드된 자료와 해당 유입 채널을 알려줘.
```

> 💡 이벤트 이름이 다르다면: `file_download` 대신 실제 이벤트 이름으로 교체하세요  
> (예: `download_catalog`, `brochure_download`)

## 주요 페이지 성과

### 핵심 전환 페이지 분석

```
지난 28일 아래 유형의 페이지 성과를 분석해줘:
- 제품/서비스 소개 페이지
- 문의하기 페이지
- 회사 소개 페이지

각 페이지별로:
- 세션 수
- 평균 체류 시간
- 이탈률
- 문의 폼 제출 수 (해당되는 경우)

어떤 페이지에서 방문자가 가장 오래 머무는지, 이유를 추론해줘.
```

### 제품·서비스 페이지 인게이지먼트

```
지난 28일 URL에 '/product' 또는 '/service' 가 포함된 페이지들의
세션 수, 평균 체류 시간, 이탈률을 비교해줘.

체류 시간이 긴 페이지와 짧은 페이지의 차이점을 분석해줘.
```

## 방문자 품질 분석

B2B에서는 '얼마나 오래 머물렀는가'가 구매 의향의 지표입니다.

### 고품질 방문자 채널 찾기

```
지난 28일 채널별로 아래 지표를 비교해줘:
- 평균 세션 시간 (분:초)
- 세션당 페이지뷰 수
- 이탈률
- 재방문율 (재방문자 비율)

단순히 세션 수가 많은 채널이 아니라,
체류 시간이 길고 이탈률이 낮은 '고품질' 유입 채널이 어딘지 분석해줘.
```

### 재방문자 행동 분석

```
지난 28일 재방문자(returning users)와 신규 방문자를 비교해줘.

표 항목:
- 구분
- 세션 수
- 평균 세션 시간
- 페이지뷰 수
- 리드 전환율

B2B 구매 사이클을 고려해 재방문자가 더 높은 전환율을 보이는지 확인해줘.
```

## 검색 유입 분석 (SEO)

B2B에서 오가닉 검색은 장기 리드 확보의 핵심입니다.

```
지난 28일 오가닉 검색 유입을 분석해줘.

1. 검색 유입 랜딩 페이지 TOP 10 (세션 수, 이탈률, 평균 체류시간)
2. 검색 유입 → 리드 전환 수 및 전환율
3. 검색 유입의 재방문율

검색을 통해 들어온 방문자 중 리드로 전환되는 비율이 어느 정도인지 알려줘.
```

## 종합 B2B 월간 리포트

```
[역할] B2B 기업 마케팅 담당자

[기간] 지난 28일 vs 그 전 28일

[목적] 리드 발생 중심의 월간 마케팅 성과 리포트

[요청] 아래 순서로 리포트를 작성해줘:

1. 핵심 지표 요약 (전월 비교)
   - 총 세션 수 / 리드(전환) 수 / 리드 전환율 / 카탈로그 다운로드 수

2. 채널별 리드 기여 표
   - 채널 / 세션 수 / 리드 수 / 리드 전환율 / 전월 대비 변화율(▲▼)

3. 주요 전환 페이지 TOP 5
   - 페이지 URL / 세션 수 / 리드 발생 수 / 평균 체류 시간

4. 방문자 품질 지표
   - 채널별 평균 체류 시간 / 이탈률 / 재방문율

5. 핵심 인사이트 3줄
   - 이달 가장 효과적인 리드 채널
   - 개선이 필요한 페이지 또는 채널
   - 다음 달 집중 액션 제안

[포맷] 한국어 헤더 / 비율 소수점 둘째 자리 / 수치 천 단위 쉼표
```

## 활용 팁

- **전환 이벤트 이름이 다를 경우**: "리드 수 대신 `contact_form` 이벤트 수로 바꿔줘" 처럼 실제 이벤트 이름을 알려주면 됩니다
- **카탈로그 다운로드가 이벤트로 안 잡히는 경우**: GA4 → 관리 → 이벤트에서 `file_download` 이벤트가 자동 수집되는지 확인하세요
- **소규모 사이트라 데이터가 적은 경우**: 기간을 `last_90_days`로 늘려서 분석하는 것을 권장합니다
- **GA4에 목표 설정이 없는 경우**: GA4 → 관리 → 이벤트 → 전환으로 표시 기능을 활용해 주요 이벤트를 전환으로 지정하세요
