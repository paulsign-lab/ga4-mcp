# 📊 GA4 → Google Sheets 연동 활용

GA4 데이터를 Google Sheets에 자동으로 기록하고  
표·그래프 형태로 시각화하는 활용법 모음입니다.

> 두 MCP가 모두 연결돼 있어야 합니다.  
> `/mcp` 명령으로 `ga4 · Connected` + `google-sheets · Connected` 확인 후 진행하세요.

---

## 먼저 알아두기: 두 MCP의 역할 분담

| MCP | 역할 |
|---|---|
| **GA4 MCP** | GA4 서버에서 데이터를 읽어옴 |
| **Google Sheets MCP** | 그 데이터를 시트에 기록하고 서식을 적용 |

Claude가 두 MCP를 자동으로 조합합니다. 복잡한 설정 없이 자연어 한 줄로 작동합니다.

---

## 기본: 데이터 조회 → 시트에 기록

### 채널별 주간 리포트를 새 시트에 저장

```
지난 7일 채널별 세션 수, 전환 수, 전환율을 GA4에서 가져와서
새 구글 시트를 만들어 '채널분석' 탭에 기록해줘.
헤더는 파란 배경(#4285f4)에 흰 글씨로 서식 적용해줘.
```

### 기존 시트에 이번 주 데이터 추가

```
내 GA4에서 지난 7일 채널별 세션 수를 가져와서
아래 스프레드시트의 '주간데이터' 시트 마지막 행에 날짜와 함께 추가해줘.

스프레드시트 URL: [여기에 URL 붙여넣기]
```

### 랜딩 페이지 분석 결과 기록

```
GA4에서 지난 28일 랜딩 페이지 TOP 20을 가져와서
(세션 수, 이탈률, 평균 체류시간 포함)
새 스프레드시트 '랜딩페이지분석'을 만들고 기록해줘.
이탈률 70% 이상인 행은 빨간 배경(#fce4ec)으로 강조해줘.
```

## 심화: 주간 종합 리포트 자동 생성

매주 한 번 실행하면 완성되는 종합 리포트입니다.

```
[역할] 디지털 마케팅 애널리스트

[작업] 지난 7일 GA4 데이터로 주간 마케팅 리포트 스프레드시트를 만들어줘.

[시트 구성] 4개 탭:

탭 1. 요약
- 핵심 지표 4개: 총 세션 / 전환 수 / 전환율 / 활성 사용자
- 전주 대비 변화율 (▲▼)
- 한 줄 인사이트

탭 2. 채널별 성과
- 채널 / 세션 / 전환 수 / 전환율 / 전주 대비 변화율

탭 3. 랜딩 페이지 TOP 10
- URL / 세션 / 이탈률 / 평균 체류시간

탭 4. 디바이스 분포
- 디바이스 / 세션 / 전환율

[서식]
- 모든 탭 헤더: 파란 배경(#4285f4) + 흰 글씨 + 굵게
- 이탈률 70% 이상 행: 주황 배경(#fff3e0)
- 숫자: 천 단위 쉼표 / 비율: % 두 자리

[완료 후] 스프레드시트 URL 공유해줘
```

## B2C 이커머스: 매출 리포트 시트 생성

```
GA4에서 지난 28일 매출 데이터를 가져와서
새 스프레드시트 '이커머스 월간 리포트'를 만들어줘.

탭 1. 매출 요약
- 총 매출 / 구매 수 / 평균 주문 금액(AOV) / 구매 전환율

탭 2. 채널별 매출
- 채널 / 세션 / 구매 수 / 매출 / 전환율
- 매출 기준 내림차순 정렬

탭 3. 제품별 성과
- 제품명 / 조회 수 / 구매 수 / 매출
- 매출 TOP 10

탭 4. 구매 퍼널
- 제품 조회 → 장바구니 → 결제 시작 → 구매 완료 각 단계 수치와 이탈률

서식: 헤더 초록 배경(#34a853), 매출 컬럼 ₩ 표시
```

## B2B 기업 사이트: 리드 리포트 시트 생성

```
GA4에서 지난 28일 데이터를 가져와서
새 스프레드시트 'B2B 리드 월간 리포트'를 만들어줘.

탭 1. 리드 요약
- 총 세션 / 리드(전환) 수 / 리드 전환율 / 카탈로그 다운로드 수

탭 2. 채널별 리드 성과
- 채널 / 세션 / 리드 수 / 리드 전환율
- 리드 수 기준 내림차순

탭 3. 주요 전환 페이지
- 페이지 URL / 세션 / 전환 수 / 평균 체류시간

탭 4. 방문자 품질 지표
- 채널 / 평균 세션 시간 / 이탈률 / 재방문율

서식: 헤더 남색 배경(#1a73e8) + 흰 글씨
```

## 기간 비교 리포트 (이번 달 vs 지난 달)

```
GA4에서 이번 달과 지난 달 데이터를 각각 가져와서
아래 URL의 스프레드시트 '월간비교' 시트에 기록해줘.

스프레드시트 URL: [URL 붙여넣기]

표 구성:
- A열: 채널
- B열: 이번 달 세션
- C열: 지난 달 세션
- D열: 변화율(%) — 수식으로 =((B2-C2)/C2)*100 형태로 입력
- E열: 이번 달 전환 수
- F열: 지난 달 전환 수

D열에 조건부 서식: 양수는 초록(#e6f4ea), 음수는 빨강(#fce4ec)
```

## 시트에 데이터 쌓기 (누적 기록)

매주 실행해서 데이터를 누적하면 추세 분석이 가능합니다.

```
GA4에서 이번 주(최근 7일) 채널별 세션 수와 전환 수를 가져와서
아래 스프레드시트의 '주간누적' 시트 마지막 행에 추가해줘.

스프레드시트 URL: [URL 붙여넣기]

추가할 열 순서:
A열: 날짜 (오늘 날짜 YYYY-MM-DD)
B열: 채널
C열: 세션 수
D열: 전환 수
E열: 전환율
```

## 그래프용 데이터 준비

Google Sheets에서 그래프를 만들 수 있도록 데이터를 최적화합니다.

```
GA4에서 지난 90일간 주별 세션 수와 전환 수를 가져와서
아래 스프레드시트의 '차트데이터' 시트에 기록해줘.

형식:
- A열: 주차 (예: 2026-W01)
- B열: 세션 수
- C열: 전환 수
- D열: 전환율

데이터 기록 후 Google Sheets에서 꺾은선 그래프 만드는 방법도 알려줘.
```

## 활용 팁

- **정기 실행**: "매주 월요일 이 프롬프트 실행" → 같은 프롬프트를 반복 실행하면 시트가 자동 업데이트됩니다
- **시트 ID 확인**: Google Sheets URL에서 `/d/` 뒤 `/edit` 앞의 문자열이 스프레드시트 ID입니다
- **그래프 직접 추가**: 현재 Sheets MCP는 데이터 기록까지 지원합니다. 그래프는 Google Sheets에서 데이터 선택 후 삽입 → 차트로 직접 만들어야 합니다
- **공유**: 완성된 시트 URL을 팀에 공유하면 실시간 협업 가능합니다
